# ChungYou Personal Portfolio

A sophisticated and minimalist single-page personal website designed for "ChungYou". The project prioritizes elegant typography, a clean aesthetic, and a real-time responsive clock within the hero section.

## Features

- **Modern Minimalist Aesthetics**: Inspired by high-end artist portfolios, utilizing a refined cream and dark-text off-white palette.
- **Dynamic Real-Time Clock**: A custom JavaScript-driven clock right below the developer's name to display local time seamlessly.
- **Micro-Animations & Texture**: Features smooth text-reveal animations, animated soft radial background blobs, and a subtle SVG noise overlay to emulate a premium "print" feel.
- **Premium Typography**: Pairs *Cormorant Garamond* (a sleek serif) for the main headers and *Outfit* (a modern sans-serif) for clean readability in the UI components.
- **Responsive Design**: Designed to scale flawlessly from large desktop monitors down to mobile breakpoints.

## Project Structure

The project has been separated into clean, distinct files for optimal code readability and maintenance:

- **`index.html`**: The main structural markup holding the hero elements, live time display, and footer navigation links.
- **`style.css`**: The core stylesheet housing the design system variables, layouts, CSS animations, and media queries.
- **`app.js`**: Client-side logic continuously fetching and updating the local time.

## How to Run locally

Simply clone the repository and open `index.html` in any modern web browser. 

```bash
git clone https://github.com/yoyokuo12-bot/4112056024DIC.git
```
Then, double-click `index.html` inside the folder. No local development server or compilation required.
